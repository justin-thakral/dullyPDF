# Sandbox UI

Standalone React + TypeScript UI for viewing PDFs and editing form fields. This is a frontend-only sandbox that mirrors the visual language of the main app while staying self-contained.

## Quick start

From the repo root (recommended):

```bash
npm run start sandbox
```

Or run it directly:

```bash
cd "sandbox UI"
npm install
npm run start
```

Vite will use the next available port (usually `http://localhost:5174` if the main frontend is already running).
