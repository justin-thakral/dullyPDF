# Styling Notes

The UI mirrors the main app color direction: black and deep navy foundations with bright blue accents, set against neutral light surfaces.

## Theme tokens

Defined in `sandbox UI/src/index.css`:

- `--ink-*`: Base text colors.
- `--blue-*` and `--sky-*`: Accent colors.
- `--surface-*`: Panel and canvas surfaces.
- `--shadow-*`: Shadow presets.

## Typography

- Body: IBM Plex Sans.
- Headings: Space Grotesk.

Both fonts are loaded from Google Fonts in `sandbox UI/src/index.css`.
