# Security Docs

This directory groups security findings by severity of fix readiness.
Use the report template format from `mcp/security-docs/SECURITY-REPORT-TEMPLATE.md`.

## Levels

- Type 1 (fixed): `mcp/security-docs/type-1-fixed`
- Type 2 (not fixed): `mcp/security-docs/type-2-notfixed`
- Type 3 (needs feedback): `mcp/security-docs/type-3-needs-feedback`

Each directory contains a `REPORT_TEMPLATE.md` that matches the required format.
